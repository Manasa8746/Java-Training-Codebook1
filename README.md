# Java Training Code Book

This repository contains a Java code book prepared during a 15-day Java training program.
The PDF includes Java concepts explained with simple programs and examples practiced during the training period.

## 📘 Topics Covered
- Introduction to Java
- Data Types and Operators
- Input and Output
- Conditional and Control Statements
- Arrays (1D and 2D)
- Strings and String Methods
- Methods (Static and Non-Static)
- Recursion
- Object-Oriented Programming (OOPS)
  - Class and Object
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Abstraction

## 📄 About the PDF
- Contains concept explanations with example programs
- Based only on topics taught during the training
- Prepared for learning, practice, and revision purposes

## 🛠️ Tools Used
- Java
- VS Code / online java compiler
- MS Word (converted to PDF)

## 🎯 Purpose
This repository is created to document my Java learning during training and to use as a reference for future practice and interviews.

Note: This repository focuses on Core Java and OOPS concepts learned during the training. Data Structures and Algorithms will be practiced consistently and documented in a separate repository in the upcoming days.
